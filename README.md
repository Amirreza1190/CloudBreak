## рҹ”¬ What is CloudBreak?

**CloudBreak** is a one-click, zero-cost VLESS + VMess proxy engine that runs entirely on **Cloudflare Workers** вҖ” deployed in under 60 seconds via GitHub Actions. It defeats Deep Packet Inspection (DPI) not by faking legitimacy, but by *being* legitimate.

> **The key insight:** Tools like V2Ray's Reality protocol *fake* a legitimate TLS certificate. CloudBreak doesn't fake anything вҖ” Cloudflare **is** a legitimate host with a **real** cert on a **real** globally-routable domain. DPI sees nothing but ordinary Cloudflare HTTPS traffic.

## рҹ”„ Key Rotation

Every workflow run:
1. **Deletes** the existing worker (fresh start)
2. **Generates** new UUID from `/proc/sys/kernel/random/uuid`
3. **Generates** new WS path via `openssl rand -hex 6`
4. **Deploys** fresh worker with new keys + new QR codes

The `/sub` subscription URL **always reflects current keys** вҖ” clients using subscriptions update automatically.

## рҹҢҗ Optional: PROXYIP Chaining

If `workers.dev` is blocked by your ISP, set the `proxy_ip` workflow input to a relay server IP you control:

```
Client вҶ’ Cloudflare Worker вҶ’ Your relay IP вҶ’ Internet
```

Two-hop routing defeats even direct `workers.dev` blocks.

```
в”Ӯ   в””в”Җв”Җ Health check                  # /health endpoint (JSON status)
в”Ӯ
в”ңв”Җв”Җ .github/
в”Ӯ   в””в”Җв”Җ workflows/
в”Ӯ       в””в”Җв”Җ deploy-vless.yml          # GitHub Actions deployment pipeline
в”Ӯ           в”ңв”Җв”Җ Key generation         # UUID + WS path from kernel entropy
в”Ӯ           в”ңв”Җв”Җ Wrangler deploy        # CF Worker deployment
в”Ӯ           в”ңв”Җв”Җ QR code generation     # Python qrcode library
в”Ӯ           в””в”Җв”Җ Step summary           # Beautiful HTML summary with QRs
в”Ӯ
в””в”Җв”Җ README.md                         # You are here
```

## рҹ”җ Security Model

- UUID stored as Cloudflare Worker env var вҖ” **encrypted at rest**
- UUID **never** stored in this repo or GitHub secrets
- Each deploy generates completely new credentials
- Random WS path вҖ” no discoverable default endpoint
- TLS terminated by Cloudflare вҖ” your device never handles raw TLS
- VLESS frames inside TLS вҖ” payload is fully opaque to DPI

## рҹ”„ ЪҶШұШ®Шҙ Ъ©Щ„ЫҢШҜ (Key Rotation)
